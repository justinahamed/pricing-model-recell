# Dynamic Pricing Model (ReCell)

## Overview
Developed a regression-based pricing model to identify key factors influencing the price of refurbished devices and support data-driven pricing strategy.

## Problem
Refurbished device markets require pricing that balances competitiveness with profitability. The goal was to determine the key drivers of price and build a predictive model.

## Approach
- Conducted exploratory data analysis (EDA)
- Built linear regression model
- Validated model assumptions (linearity, multicollinearity)
- Performed feature selection to improve model performance

## Results
- Identified key drivers such as device specifications and condition
- Generated insights to support pricing optimization and revenue improvement

## Tech Stack
Python, pandas, scikit-learn, matplotlib
